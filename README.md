# Birthday Slideshow 🎂✨

A cinematic, emotional, and modern birthday slideshow web application built with **React**, **Tailwind CSS**, and **Framer Motion**.

## Features

- **Dynamic Image Loading**: Automatically shuffles and displays images from `public/img/`.
- **Cinematic Transitions**: Random cycle between Fade, Zoom, Slide, and Blur.
- **Ken Burns Effect**: Subtle panning and zooming for every photo.
- **Background Music**: Integrated player with controls.
- **Emotional Overlays**: Beautifully animated birthday wishes.
- **Responsive**: Optimized for both mobile and large screens.

## Setup & Usage

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd birthday-slideshow
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Add your media**:
   - Place photos in `public/img/` (JPEG format).
   - Place your music track as `public/music.mp3`.

4. **Launch the app**:
   ```bash
   npm run dev
   ```
